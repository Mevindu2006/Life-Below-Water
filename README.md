# 🌊 Life Below Water

## 📌 Project Overview

**Life Below Water** is a web-based project focused on raising awareness about **coral ecosystems** and the importance of protecting marine life. The website educates users about coral reefs, environmental threats, and encourages actions such as beach cleanups to preserve ocean biodiversity.

---

## 🎯 Objectives

* Promote awareness about coral reefs and marine ecosystems
* Highlight environmental issues affecting ocean life
* Encourage community participation in beach cleanups
* Educate users through interactive and visually appealing content

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout design

---

## ✨ Features

* 📱 **Responsive Design** – Works on mobile, tablet, and desktop devices
* 🎨 **Animations** – Smooth visual effects to enhance user experience
* 🧾 **Interactive Forms** – User input for engagement (e.g., cleanup participation)
* 🌊 **Educational Sections** – Information about coral reefs and ocean conservation
